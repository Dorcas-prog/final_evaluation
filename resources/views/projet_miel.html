Documentation du projet "Produits_miel"

Introduction :

Le projet "Produits_miel" est une application web conçue pour faciliter la gestion et la commande de produits à base de miel. Ce document présente les objectifs, les fonctionnalités, la structure, les technologies utilisées, ainsi que les étapes pour installer et exécuter le projet.

1. Objectif du projet

L'objectif principal de ce projet est de fournir une plateforme permettant aux utilisateurs de découvrir, commander et en apprendre davantage sur les produits à base de miel.

2. Fonctionnalités principales

  • 2.1.Gestion des commandes :

    les utilisateurs peuvent passer des commandes pour différents produits, tels que

    2.1.1. Le miel simple,

    2.1.2. Le miel cannelé,

    2.1.3. Le miel sorti dirrectement de la ruche et

    2.1.4. Le savon fabriqué à base de miel

  • 2.2Affichage des produits

    . Une liste des produits disponibles est affichée avec leurs descriptions et prix plus haut.

    . Non seulement ça, le miel est un aliment énergétique qui opère très miraculeusement dans tous les domaines. Il aide à :

    - guérir la tout ;

    - guérir les plaies ;

    - traiter la peau toutefois en gardant le teint tel qu'il est. Il fait beaucoup d'autres choses.

  • 2.3. Contact

    Les utilisateurs peuvent contacter l'entreprise BelRich via les informations fournies ; soit à travers les contacts ou soit par email.

3. Structure du projet

Le projet suit une architecture MVC (Model-View-Controller) typique de Laravel :

  • 3.1. Modèles : Gestion des données et des interactions avec la base de données.
  • 3.2. Vues : Fichiers Blade pour afficher les pages HTML.
  • 3.3. Contrôleurs : Gestion de la logique métier et des requêtes utilisateur.

4. Technologies utilisées

  • 4.1. Laravel : Framework PHP pour le développement backend.
  • 4.2. Bootstrap : Framework CSS pour le design responsive.
  • 4.3. MySQL : Base de données pour stocker les informations.

5. Installation

Pour installer et exécuter ce projet localement, il faut :

  1. 5.1. Clonez le dépôt : git clone [URL]
  2. 5.2. Installez les dépendances PHP : composer install
  3. 5.3. Installez les dépendances front-end : npm install
  4. 5.4. Configurez le fichier .env avec vos informations de base de données.
  5. 5.5. Lancez le serveur : php artisan serve
